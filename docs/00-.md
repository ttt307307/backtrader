## 2. 安装依赖包
```
# 安装pipreqs
pip install pipreqs

# 生成依赖文件
# 进入根目录，并以utf8的格式去检测文件中的模块和版本
pipreqs ./ --encoding=utf-8

# 安装依赖包requirements.txt
pip install -r 
```
